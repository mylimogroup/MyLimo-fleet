# MyLimo Fleet

A modern fleet management system for chauffeur and limousine companies. Built with Next.js, TypeScript, Tailwind CSS, and the App Router.

## Features (planned)

- **Fleet Overview** — Real-time vehicle tracking, availability, and maintenance
- **Driver Management** — Chauffeur schedules, assignments, and compliance
- **Bookings & Dispatch** — Reservation coordination and route optimization
- **Analytics & Reports** — Utilization, revenue, and performance insights

## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [React 19](https://react.dev/)

## Getting Started

Install dependencies (if not already done):

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Scripts

| Command         | Description              |
| --------------- | ------------------------ |
| `npm run dev`   | Start development server |
| `npm run build` | Create production build  |
| `npm run start` | Start production server  |
| `npm run lint`  | Run ESLint               |
